# Portfolio v6 🚀

Welcome to the 6th iteration of my personal portfolio. This version represents a complete redesign focused on a premium, modern aesthetic with a high-performance technical foundation.

**Designed with ❤️ using Claude Design.**  
**Live Demo:** [saiii.in](https://saiii.in)

---

## ✨ Features

- **Modern Redesign:** A sleek, premium UI built with a focus on typography and clean layouts.
- **AI-Powered Chat:** Integrated interactive chat widget powered by OpenAI for real-time engagement.
- **Dynamic Projects Showcase:** Detailed categorization and display of my work across various tech domains.
- **Automated Blog:** Technical insights and bi-weekly digests managed through a robust data layer.
- **SEO Optimized:** Full SEO support with dynamic meta tags, sitemaps, and optimized performance metrics.
- **Serverless Backend:** Leveraging Vercel Functions and Supabase for a scalable, data-driven experience.

## 🛠️ Tech Stack

- **Frontend:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **State & Routing:** [React Router](https://reactrouter.com/)
- **Backend/Database:** [Supabase](https://supabase.com/) & [PostgreSQL](https://www.postgresql.org/)
- **AI Integration:** [OpenAI API](https://openai.com/)
- **Deployment:** [Vercel](https://vercel.com/)
- **Icons:** [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sairam3824/Portfolio-v6.git
   cd Portfolio-v6
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env` file based on `.env.example` and add your Vercel/Supabase/OpenAI credentials.

4. **Start the development server:**
   ```bash
   npm run dev
   ```
